baidupanZipWiewer
=================
用百度网盘在线浏览漫画


# v2.1更新


* 实现了app端直接登陆百度账号
* 内置实现了app端浏览百度网盘文件
* 实现了直接从app端解析zip文件
* 摆脱了对服务器端http://ddddddd.jd-app.com/blog/article/baidupanZipWiewer2/的依赖（保留了二维码同步百度session的功能）
* 代码重构
* bug消除
* UI美化
* 使用中文


# features

* picture zoom&drag
* Cache
* password protect
* qrcode scan sysnc session from web
